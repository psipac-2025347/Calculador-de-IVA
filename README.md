# Guia para usar la calculadora de IVA
## Uso
En la aplicación tienes que ejecutarla con en la termimal ya que la aplicación no tiene ninguna vista por ahora
** los pasos son **: 
- pnpm build
- pnpm start

En la aplicación tienes que elegir cuantos montos vas a poner dentro de la aplicación. E ingresas el numero de montos, Correspondiendo a ese numero
se generaran los espacios para poder ingresar los montos en los cuales tu tienes rellenar cada uno.

al acabar se te mostrará el subtotal, El IVA, el precio sin IVA y el total.Y acaba el flujo de la aplicación

## Tecnologias
- pnpm
- TypeScript
- Visual Studio Code
- node.js
